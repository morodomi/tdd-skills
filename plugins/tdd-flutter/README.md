# tdd-flutter

Flutter/Dart quality tools for TDD workflow with Claude Code.

## Installation

```bash
/plugin install tdd-flutter@tdd-skills
```

## Skills

| Skill | Description |
|-------|-------------|
| flutter-quality | Flutter quality checks (dart analyze, dart format, flutter test) |

## Tools

| Tool | Command |
|------|---------|
| Analyze | `dart analyze` |
| Format | `dart format .` |
| Test | `flutter test` |
| Coverage | `flutter test --coverage` |

## Usage with Core Plugin

Combine with tdd-core for full TDD workflow:

```bash
/plugin install tdd-core@tdd-skills
/plugin install tdd-flutter@tdd-skills
```
